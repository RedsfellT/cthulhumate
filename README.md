## Installer l'app sur chaque appareil

### 💻 PC (Chrome ou Edge)
1. Ouvre **https://RedsfellT.github.io/cthulhumate/**
2. Clique l'icône **📥** dans la barre d'adresse
3. Clique **Installer**

### 📱 Android (Chrome)
1. Ouvre **https://RedsfellT.github.io/cthulhumate/**
2. Menu **⋮** → **Ajouter à l'écran d'accueil**

### 🍎 iPhone / iPad (Safari uniquement)
1. Ouvre **https://RedsfellT.github.io/cthulhumate/**
2. Bouton **Partage** ↑ → **Sur l'écran d'accueil**

---

## Session WiFi (à la table de jeu)

1. Lance **`start.bat`** sur le PC du Gardien
2. Le terminal affiche l'IP et un QR code
3. **Première fois par appareil** : ouvre `https://[IP]:3000` dans le navigateur → *Avancé* → *Continuer*
4. Dans l'app → **Gardien** → **📡 Session** → entre `[IP]:3000` → *Se connecter*

---

## Mettre à jour l'app

Double-clic sur **`deploy.bat`** → mis à jour sur tous les appareils en ~2 min.
